# AI Exam Grader - Project Instructions

## 테스트 우선 개발 (TDD)
- 새로운 기능 구현이나 버그 수정 시, **단위 테스트와 통합 테스트를 먼저 작성**한 뒤 구현 코드를 작성한다.
- 테스트 프레임워크: vitest
- 테스트 실행: `npm test` (watch 모드), `npm run test:run` (단일 실행)

## 과거 이슈 참고
- 유사한 문제를 만나면 `docs/lessons-learned/`의 문서를 먼저 확인한다.
- 해결된 이슈의 원인, 해결 방법, 교훈이 정리되어 있다.
